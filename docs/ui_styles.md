# UI Styles (Dark)

## Category List (QListWidget)

- Background: `#2b2b2b`
- Selection: blue `#3d5a80` + white text
- Hover: gray `#3a3a3a`
- Text: light gray `#e0e0e0`
- Padding: 4px
- Item margin: 2px vertical
- View spacing: 0 (can be set to 6–10 for more breathing room)

## Settings Area (QStackedWidget)

- Background: `#333333`
- Border: `#3d3d3d`
- GroupBox: background `#3a3a3a`, border `#4a4a4a`, radius 6

## Tuning Tips

- Increase view spacing to 8–10 if preferred visually
- Harmonize corner radius with global UI style
- Use tooltips to explain parameters
